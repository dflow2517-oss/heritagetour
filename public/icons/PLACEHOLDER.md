# PWA Icons

Add two PNG files here before deploying:

- `icon-192.png` — 192×192 px
- `icon-512.png` — 512×512 px

Both should use the burnt-orange background (#b8491c) with a cream "H" or
the commission seal. The 512 will also serve as the maskable icon.

Until real icons are added the PWA will still install; it just uses the
browser's default placeholder icon.
